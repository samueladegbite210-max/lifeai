# LifeAI - Global AI Life Assistant

A simple but powerful personal AI assistant to help you:

- Manage daily tasks
- Track your budget
- Set and monitor goals  
- Build better habits
- Get motivational insights every day

**How to use:**
1. Open → [LifeAI](https://samueladegbite210-max.github.io/lifeai/)
2. Tap "Add to Home Screen" to install it like a real app
3. Start adding tasks and goals

Completely free. Works offline after first visit.
